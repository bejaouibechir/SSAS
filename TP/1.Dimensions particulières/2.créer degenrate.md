## Ajouter une dimension dégénérée (Degenerate Dimension) dans SSAS (procédure)

### 1) Choisir la colonne à dégénérer

Exemples dans ta table **FactProductSales** :

- SalesInvoiceNumber

- InvoiceLineNumber  
  (une “dimension dégénérée” = identifiant métier dans la table de faits)

---

### 2) Ajouter la colonne dans le groupe de mesures

Dans le cube :

- Onglet **Structure de cube**

- Dans **Fact Product Sales** :
  
  - Repère la section **Dimensions** ou **Measures / Measure Group**
  
  - Ajoute l’attribut correspondant (SalesInvoiceNumber)

---

### 3) Créer une dimension basée sur la table de faits

- Dans **Dimensions** (Solution Explorer) :
  
  - Clique droit → **New Dimension**
  
  - Source : **Existing table**
  
  - Table principale : **dw.FactProductSales**
  
  - Clé : **SalesInvoiceNumber**
  
  - Attributs :
    
    - SalesInvoiceNumber (obligatoire)
    
    - InvoiceLineNumber (optionnel)
  
  - Finish

---

### 4) Ajouter la dimension au cube

- Ouvre le cube

- Onglet **Structure de cube**

- Clique droit sur **Dimensions** → **Add Cube Dimension**

- Choisis la dimension que tu viens de créer
  
  ![](D:\Formation%20BI\SSAS\Labs\0.%20Création%20Datawarehouse\images\18.png)

---

### 5) Lier la dimension au groupe de mesures

- Onglet **Utilisation de la dimension**

- Ligne : ta dimension dégénérée (ex: Dim SalesInvoice)

- Colonne : **Fact Product Sales**

- Type : **Normale**

- Granularité : **SalesInvoiceNumber**

---

### 6) Déployer / Tester

- Deploy

- Dans le navigateur :
  
  - Mets SalesInvoiceNumber en lignes
  
  - Mets une mesure (TotalAmount, Quantity, Count, etc.)
