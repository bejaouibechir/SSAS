# Créer un groupe de mesures



Voici un tutoriel **bref** et pas à pas (style Microsoft Walkthrough) pour **créer un groupe de styles** (appelé **Dossier d’affichage** ou **Display Folder**) dans SSDT 2022 pour un modèle Tabulaire, puis comment l’utiliser — en français :

1. **Ouvrez** votre projet Tabulaire dans **SQL Server Data Tools (Visual Studio 2022)**  
   → **Fichier > Ouvrir > Projet/Solution** → sélectionnez votre projet .bim

2. Dans **Explorateur de solutions**, cliquez droit sur **Model.bim** → **Afficher le concepteur**

3. Passez en vue **Grille de mesures** (en bas) ou sélectionnez la table qui contient les **mesures** / **colonnes calculées** que vous voulez organiser

4. Sélectionnez une ou plusieurs **mesures** (ou colonnes calculées) dans la grille  
   → maintenez **Ctrl** pour en sélectionner plusieurs

5. Dans la fenêtre **Propriétés** (généralement à droite), trouvez la propriété **Dossier d’affichage** (Display Folder)

6. Saisissez un nom de dossier (ou plusieurs niveaux avec `\`)  
   Exemples :  
   
   - `Ventes\Marges`  
   - `Indicateurs\KPI Année en cours`  
     → Validez avec **Entrée**

7. Répétez l’opération pour d’autres mesures / colonnes afin de créer plusieurs **groupes** logiques  
   (ex. : `Ventes\Chiffre d’affaires`, `Ventes\Quantités`, `Clients\Segmentation`)

8. **(Optionnel – très recommandé)** Créez aussi des dossiers pour les **colonnes** des tables de dimension :  
   → cliquez sur une table dans l’explorateur → sélectionnez des colonnes → **Dossier d’affichage** dans Propriétés

9. **Enregistrez** le modèle (**Ctrl + S**)

10. **Testez** immédiatement l’organisation :  
    → **Modèle > Analyser dans Excel**  
    → dans la liste des champs Excel, vous verrez maintenant les **dossiers** créés avec les mesures / colonnes regroupées dedans

11. **Déployez** le modèle pour que les utilisateurs profitent des dossiers :  
    → clic droit sur le projet → **Déployer**  
    → connectez-vous au serveur **Analysis Services** → terminez  
    → les dossiers d’affichage apparaîtront dans Power BI, Excel, etc.
